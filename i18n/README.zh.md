# 🎄 画个圣诞帽

「画个圣诞帽」是一款轻量有趣的微信小程序，可获取用户头像并一键叠加圣诞帽贴图。无需登录、无需繁琐配置，打开即可使用，专注营造节日氛围。

## ✨ 功能介绍

核心功能：
- 获取用户微信头像（需授权）
- 在头像上自动叠加圣诞帽贴图
- 实时预览生成效果
- 将生成头像保存到本地相册

❗ 本小程序不包含社交、上传、分享或账号体系，也不收集个人信息。

## 🧩 使用流程
1. 打开微信小程序「画个圣诞帽」
2. 授权获取微信头像
3. 点击「一键加圣诞帽」
4. 预览效果并保存头像

## 🔐 隐私说明
- 头像仅在本地 Canvas 处理
- 不上传服务器
- 不存储、不分析、不共享任何用户数据
- 无需手机号，无需登录
