# 🎄 画个圣诞帽

「画个圣诞帽」是一款轻量有趣的微信小程序，可一键为头像戴上圣诞帽。无需登录、无需复杂配置，打开即用，快速营造节日氛围。

## ✨ 功能介绍

![](/public/15.png)

核心功能：

- 获取微信头像（需授权）
- 自动叠加圣诞帽贴图
- 实时预览生成效果
- 一键保存到本地相册

❗ 无社交、无上传、无账号体系，不采集个人信息。

## 🧩 使用流程

![](/public/qr_code.png)

1. 扫描上方二维码，或在微信搜索「画个圣诞帽」
2. 选择头像并点击生成
3. 预览并保存生成的头像

## ⚠️ 注意事项

算法偶尔可能生成出人意料的效果
