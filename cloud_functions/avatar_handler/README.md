### 头像处理函数

1. 获取用户上传的文件
2. 保存用户上传的文件,拿到url
3. 使用url在阿里千问生成图片临时url
4. 下载临时url图片
5. 保存到腾讯OSS

#### 环境配置项

| 名称              | 说明               |
| ----------------- | ------------------ |
| DASHSCOPE_API_KEY | 阿里云API_KEY      |
| DASHSCOPE_URL     | 阿里云大模型地址   |
| PIC_PROMPT        | 阿里云大模型提示词 |
| CLOUD_ENV         | 腾讯云ID           |
